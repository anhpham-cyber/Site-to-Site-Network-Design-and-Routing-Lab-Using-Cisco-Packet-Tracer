# Site-to-Site-Network-Design-and-Routing-Lab-Using-Cisco-Packet-Tracer
This Cisco Packet Tracer lab simulates a site-to-site network between Calgary and Edmonton. It covers the configuration of PCs, switches, and routers to enable cross-site communication. Key focus areas include IP addressing, Layer 2 switching, Layer 3 routing (Static, RIP, OSPF, EIGRP), and network troubleshooting.
